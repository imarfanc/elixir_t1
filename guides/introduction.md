# Introduction

Welcome to the ElixirT1 documentation!

This project is a Phoenix application designed to demonstrate best practices and modern web development techniques with Elixir.

## Getting Started

To get started with the project, make sure you have Elixir and Erlang installed.

1.  Clone the repository.
2.  Run `mix setup` to install dependencies.
3.  Start the server with `mix phx.server`.

## Features

*   **Phoenix LiveView**: Real-time interactive UI without writing custom JavaScript.
*   **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
*   **ExDoc**: Documentation generation for Elixir projects.
